# 内置表情目录（已弃用）

> **现状（v0.4.1 起）**：Magpie **不再随包提供内置精选表情**，也不在首次启动拷贝任何内置素材。
> 表情库默认为空，完全由用户通过剪贴板图片条目右键「添加到表情包」自行添加，存入
> `%APPDATA%\app.magpie\emojis\user\`。

## 历史背景

早期版本曾计划随安装包打包一批内置精选表情，并在首启拷贝到用户数据目录。该机制
（`copy_builtin_emojis_if_needed` 与 `tauri.conf.json` 的 `bundle.resources`）已在 v0.4.1 移除：

- 产品方向调整为「表情库默认为空，由用户自由添加」，更轻量、无版权顾虑。
- 因此本目录不再被打包，也不再被任何代码读取。

## 本目录为何保留

仅保留空目录结构（`.gitkeep`）与本说明，避免历史引用与文档链接 404。如需彻底清理，
可在后续版本连同本目录一并删除。
